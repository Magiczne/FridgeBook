- User should be able to access the application through Internet.
- System should have a functionality of creating an account for every user.
- User should be able to add notes, which will be visible for other users.
- User should be able to mark notes as read.
- User should have a possibility of being notified about already read notes.
- User should be able to remove the note from the fridge so that it won't be visible nor accessible in the future.
- User should be able to access his notes history.
- User should be able to edit his notes.
- User should be able to download notes which are currently available
- User should be able to create virtual fridge on which he could share notes with others.
- User should be able to synchronize notes with google keep
- Notes should have a date of adding
- Admin should be able to manage users
